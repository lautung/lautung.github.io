# 第一个练习应用

第一个练习应用不追求复杂功能，只要能把项目创建、界面展示、状态更新和本地调试连起来。

## 目标

做一个“学习清单”应用：

- 显示三条学习任务。
- 点击任务后切换完成状态。
- 页面顶部显示已完成数量。
- 使用 Compose 编写界面。

## 数据模型

```kotlin
data class StudyTask(
    val title: String,
    val completed: Boolean = false,
)
```

## 验收标准

- 应用能在模拟器启动。
- 初始页面能看到三条任务。
- 点击任务后完成状态会变化。
- 已完成数量会同步更新。

## 复盘记录

完成后记录三件事：

1. 哪一步最顺利。
2. 哪一步卡住最久。
3. 下一次练习要补哪块知识。
