# 高级 UI 导读

高级 UI 主要覆盖传统 View 体系中的界面能力。即使项目使用 Compose，理解 View、RecyclerView、自定义 View 和动画仍然有价值，因为大量老项目、三方库和系统控件都建立在 View 体系上。

## 学习顺序

1. 先理解 View 与 ViewGroup 的测量、布局和绘制。
2. 学习 RecyclerView 的 Adapter、ViewHolder 和列表性能。
3. 学习动画、手势和触摸事件分发。
4. 最后进入自定义 View，理解如何自己绘制和处理交互。

## 重点能力

- 能读懂 XML 布局和传统 View 代码。
- 能优化复杂列表的滚动体验。
- 能写基础自定义 View。
- 能区分传统 View UI 与 Compose UI 的适用场景。
