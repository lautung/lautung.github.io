# 动画与手势

动画让界面变化更自然，手势让用户能直接操作内容。两者都应该服务交互，不要为了炫技而增加负担。

## 动画类型

- 属性动画：改变 View 的属性，例如透明度、位移、缩放。
- 过渡动画：页面或布局变化时的过渡效果。
- 帧动画：逐帧播放图片，使用场景较少。

## 触摸事件

传统 View 的触摸事件会经过分发、拦截和消费：

- `dispatchTouchEvent`
- `onInterceptTouchEvent`
- `onTouchEvent`

## 学习建议

先从简单属性动画开始。只有在处理复杂嵌套滑动、自定义控件或手势冲突时，再深入事件分发。
