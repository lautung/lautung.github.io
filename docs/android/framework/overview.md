# Framework 导读

Android Framework 位于应用与系统底层之间，提供 Activity、Window、Package、Notification 等系统能力。学习 Framework 有助于理解系统行为和复杂问题。

## 学习范围

- 应用进程与系统进程的关系。
- 系统服务如何暴露能力。
- Binder/IPC 如何连接进程。
- Activity、Window、ViewRoot 等关键流程。

## 学习建议

Framework 难度较高，建议在熟悉应用开发后再进入。学习时从具体问题入手，例如 Activity 启动、触摸分发或窗口显示。
