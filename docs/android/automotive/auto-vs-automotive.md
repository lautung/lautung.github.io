# Android Auto 与 Automotive

Android Auto 是手机应用在车机上的投射体验，Android Automotive OS 则是直接运行在车机上的操作系统。

## 对比

- Android Auto：应用主要运行在手机上。
- Android Automotive OS：应用可以安装在车机系统中。
- 两者都需要遵守车载安全和分心限制。

## 学习建议

如果目标是普通手机应用支持车载场景，先看 Android Auto。如果目标是车机系统应用，再学习 Automotive OS。
