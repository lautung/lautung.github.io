# Android 车载导读

Android 车载开发主要包括 Android Automotive OS 和 Android Auto。两者名字相近，但定位不同。

## 两个概念

| 名称 | 说明 |
| --- | --- |
| Android Automotive OS | 运行在车机上的 Android 系统。 |
| Android Auto | 手机应用投射到车机屏幕的体验。 |

## 学习重点

- 车载交互要减少分心。
- UI 信息密度要克制。
- 应用类型、模板和审核要求更严格。

## 学习建议

先理解 Android Auto 与 Automotive 的差异，再根据目标平台学习对应 API 和设计规范。
