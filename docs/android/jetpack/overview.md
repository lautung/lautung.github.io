# Jetpack 组件导读

Jetpack 是一组帮助开发者更稳定构建 Android 应用的库。它覆盖生命周期、数据、导航、后台任务、界面和测试等领域。

## 常用组件

| 组件 | 作用 |
| --- | --- |
| Lifecycle | 感知生命周期，减少资源泄漏。 |
| ViewModel | 保存页面状态，适应配置变更。 |
| Navigation | 管理页面跳转和返回栈。 |
| Room | 本地数据库访问层。 |
| DataStore | 保存轻量配置数据。 |
| WorkManager | 可靠执行后台任务。 |

## 学习建议

不要一次性学习所有 Jetpack 组件。先掌握 Lifecycle、ViewModel、Navigation，再根据数据和后台任务需求学习 Room、DataStore、WorkManager。
